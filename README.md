📦 Download kits via [@WaterkoofAI_Bot](https://t.me/WaterkoofAI_Bot) — use `/kithub` to browse and unlock kits with points.

## How to Use a Kit

1. Open [@WaterkoofAI_Bot](https://t.me/WaterkoofAI_Bot)
2. Use `/kithub` to browse available kits
3. Use `/download [project name]` to unlock a kit with points
4. Follow the kit instructions

## How to Earn Points

- `/invite` — +30 pts per friend you invite
- `/rate` — +3 pts per kit rating (max 3/day)
- `/submit` — +150 to +400 pts for approved kits
